# Full Stack Open

My working repository for the [Full Stack Open](https://fullstackopen.com) course.

## Course Overview

Building modern web applications with JavaScript. The course covers:

- **React** - Building single-page applications
- **Node.js & Express** - RESTful API development
- **MongoDB** - Database integration
- **Testing** - Unit and integration testing
- **User authentication & authorization**
- **GraphQL** - Alternative to REST
- **TypeScript** - Type safety for JavaScript
- **CI/CD & deployment**

Each part of the course will have its own directory with exercises and projects.

## Stack

- JavaScript/TypeScript
- React
- Node.js
- Express
- MongoDB
